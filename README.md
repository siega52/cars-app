**Cars Manager** — современное SPA-приложение для управления автопарком с возможностью просмотра, сортировки, редактирования и удаления автомобилей. Проект разработан на React + TypeScript с использованием Vite и соответствует всем требованиям современной веб-разработки.

## Особенности

- **Полностью адаптивный дизайн** — корректное отображение на мобильных (≤480px), планшетах (≤768px) и десктопах
- **Стеклянный эффект (Glassmorphism)** — современный интерфейс в стиле iOS 18/macOS Sonoma
- **Интерактивная карта** — нативная интеграция Яндекс.Карт с кластеризацией и цветовой индикацией
- **Полный CRUD** — создание, чтение, обновление и удаление автомобилей с локальной эмуляцией API
- **Сортировка** — по году выпуска и цене (возрастание/убывание)
- **БЭМ методология** — чистая и поддерживаемая структура CSS
- **Семантическая верстка** — использование HTML5 тегов (без table)
- **Высокая производительность** — оптимизация с помощью Vite и useMemo
- **Hover-эффекты** — интерактивные карточки автомобилей
- **Обратная связь** — уведомления об успешных операциях и ошибках
